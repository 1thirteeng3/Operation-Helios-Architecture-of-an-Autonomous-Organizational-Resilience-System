# Aprendizado de Máquina e MLOps

Aqui são armazenados notebooks, scripts de treinamento e artefatos de modelos utilizados na fase de MLOps/LLMOps da simulação.  Os experimentos são versionados com MLflow e integrados aos pipelines de CI/CD.  Utilize esta pasta para explorar métricas de desempenho, detecção de drift e fairness, bem como para armazenar checkpoints de modelos.

Nesta versão inicial o diretório está vazio.  Durante o desenvolvimento você irá adicionar notebooks (Jupyter/IPython) e scripts específicos de treinamento e inferência.
