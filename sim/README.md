# Simulações

Esta pasta contém modelos de simulação para estudar o comportamento do sistema sob falhas, ataques e demandas variáveis.  Os modelos podem ser implementados em Python utilizando bibliotecas como NetworkX ou SimPy, ou ainda em ferramentas externas como AnyLogic e NetLogo.  Utilize este espaço para desenvolver simulações de Monte Carlo, avaliações de risco e propagação de falhas em cascata.
